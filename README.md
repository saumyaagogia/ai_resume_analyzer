# AI Resume Analyzer

An AI-powered web application that analyzes resumes and provides intelligent feedback using OpenAI’s GPT models. The system extracts text from PDF resumes, evaluates candidate profiles, and generates actionable insights to improve resume quality.

## 🚀 Features

* 📄 Upload and parse PDF resumes
* 🤖 AI-based analysis using OpenAI API
* 🧠 Summary, key skills extraction, and suggestions
* 📊 ATS-style scoring with category-wise breakdown
* 📈 Visual representation using charts and progress bars
* ⚡ Interactive UI built with Streamlit

## 🛠️ Tech Stack

* Python
* Streamlit
* OpenAI API
* PyPDF2
* Pandas

## 🎯 Use Case

Helps students and job seekers evaluate their resumes, identify missing skills, and improve their chances of passing Applicant Tracking Systems (ATS).

## 🔒 Environment Setup

Create a `.env` file and add:
OPENAI_API_KEY=your_api_key_here

## ▶️ Run Locally

pip install -r requirements.txt
streamlit run main.py
